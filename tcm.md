## <a id="脏象">2、脏象（基础）</a>

### <a id="脏象/五脏">2.1 五脏</a>

* <a id="脏象/五脏/心">心</a>：
    * 生理与病理：
        * 心主血脉：推动血液在脉管内运行和营养全身。
        * 心藏神（亦称心主神明）：
            * 神的涵义：精神、思维活动以及一切生命活动。
            * 神的生成：源于父精母血，且依赖后天水谷精气的滋养。
            * 五脏皆藏神：心藏神、肝藏魂、肺藏魄、脾藏意、肾藏志，
            但**心之神志**居于**首要**地位，是**一切精神意识、思维活动**的**总司**。
            * 心主神明与心主血脉是彼此联系、互相影响的。
        * 心主汗液：心之所藏，在内者为血，在外者为汗，汗者心之液也。
        夺汗者无血，夺血者无汗。
    * 与躯体官窍的关系：
        * 其华在面
        * 开窍于舌（尤其是舌尖）
        * 与小肠相表里

* <a id="脏象/五脏/肺">肺</a>：
    * 生理与病理：
        * 肺主气：司呼吸（肺主呼，肾主纳；“肺为气之主，肾为气之根”），
        以及生成真气（真气 = 自然界清气 + 脾吸收的水谷之气 + 肾之精气 = 宗气 + 先天精气）。
        * 肺朝百脉，助心行血：百脉（经脉）会合于肺，宗气在其中运行，推动血液的循环。
        * 肺主通调水道：“肺为水之上源”，通调水道的机制主要依赖肺气的宣发和肃降（详见后面），
        失调则水湿停聚而为痰饮，故善治痰饮者必先治气，气行则水行，痰饮自消。
        * 肺主声：声音同时与肺、肾有关，有“肺为声音之门，肾为声音之根”，
        且要注意肺的病理特性在声音方面表现为“金实则无声，金破亦无声”。
    * 与躯体官窍的关系：
        * 肺主皮毛：皮毛为一身之表，包括皮肤、汗腺和毛发等组织，生理功能有：
            * 调节水液代谢：见前面通调水道一节。
            * 调节呼吸：注意皮肤的汗孔也有散气作用。
            * 调节体温：卫气司汗孔的开合，有调节体温的作用。
            * 屏障作用：卫气能温养皮毛，有护卫肌表、抵御外邪的作用。
        * 开窍于鼻。
        * 与大肠相表里。
    * 生理特性：
        * 肺为娇脏，畏寒畏热：容不得丝毫异物，又不耐寒热，在五脏病变中，仅肺有表证。
        * 肺主宣发和肃降：
            * 宣发：宣布（输布）、发散，能使（清）水液布散到周身，特别是到皮毛，由汗孔排泄。
            * 肃降：肃为清肃、宁静，降为下降，能使无用（浊）的水液下归于肾而输于膀胱，
            排出体外。

* <a id="脏象/五脏/脾">脾</a>：
    * 生理与病理：
        * 脾主运化：
            * 运化水谷精微：这种去粗取精的功能若正常强健，则称为脾气健运，否则称为脾失健运。
            * （运）输（敷）布水液：又称运化水湿。“脾为生痰之源”、“诸湿肿满，皆属于脾”，
            若脾失健运则会生湿，且两者互为因果。
        * 脾能益气：真气的生成需要水谷精微的参与，所以脾虚与气虚是互相影响，
        治疗上也经常是补气药和健脾药同用。
        * 脾主生血统血：
            * 生血：脾运化的水谷精微是血液的重要物质基础，所以脾是一个重要的造血器官。
            * 统血：脾气能统摄周身血液，使之正常运行而不溢于血管之外。若脾气不足，
            则血失统摄而妄行于外，发生各种出血现象，例如长期慢性皮下出血、便血、
            月经过多等。
        * 脾主交通心肾：令心火下降，温煦肾水使之不寒；肾水上升，使心火不上炎。
        若脾阳不运，则心肾不交，会导致心悸怔忡、失眠多梦、腰膝酸软、遗精等问题。
        * 从解剖学的角度，包括了脾和胰，但根据生理和病理的表现，则包括了现代医学的消化、
        血液、植物神经、代谢、免疫等综合功能系统。
    * 与躯体官窍的关系：
        * 脾主肌肉、四肢
        * 开窍于口，其华在唇
        * 与胃相表里
    * 生理特性：
        * 脾主升清
        * 喜燥恶湿

* <a id="脏象/五脏/肝">肝</a>：
    * 生理与病理：
        * 肝主藏血：贮藏血液（休息时流回肝脏）和调节血量（劳作时运送到全身各器官）。
        若病变，则肝不藏血，进而导致肝血不足，最后两者循环。
        * 肝主疏泄：
            * 调节精神情志活动：正常则情绪乐观、心情舒畅，否则精神抑郁、情志失调。
            疏泄无能称为肝气郁结（郁郁不乐，多疑善虑），太过称为肝气横逆（急躁易怒，
            失眠多梦等）。
            * 促进消化吸收：
                * 影响脾胃的消化吸收：直接影响脾胃的升降——肝升则脾亦升，
                胆（胆与肝互为表里）降则胃亦降。肝失疏泄可导致肝脾/胃不和，
                又称肝气犯脾/胃。
                * 影响胆汁的分泌排泄：胆汁来源于脾胃化生的水谷精微，在肝内形成，
                贮存于胆囊，最后输送到小肠以助运化。
            * 维持气血的正常运行。
            * 疏利三焦，通调水道。
        * 以上两者的关系：
            * 肝血为体，是疏泄的物质基础。
            * 疏泄为用，是藏血的功能表现。
            * 所以，肝是体阴而用阳。
            * 肝病的特点：肝用（肝阳、肝气）常有余，肝体（肝阴、肝血）常不足。
    * 与躯体官窍的关系：
        * 肝主筋：注意除了一般的筋，还包括生殖系统的功能（“前阴者，宗筋之所聚”），
        所以，男子阳痿、滑精，女子白带等症，中医也从肝论治。
        * 其华在爪：爪为筋之余。
        * 开窍于目
    * 生理特性：
        * 曲直刚柔：
            * 以木的曲直来比喻肝的喜条达、恶抑郁之性。
            * 肝为刚脏，“肝为将军之官”，刚强之性体现在肝气上：太过则性急善怒，
            不足则恐惧胆怯。肝发病时，常影响上下左右，欺强凌弱，故有“肝为五脏之贼”之说。
            * 肝体宜柔，必须依赖阴血的滋养，治疗上亦常用[白芍](#白芍)之类的药物来柔肝，
            以保护肝阴阴血为要。
        * 体阴用阳：以上。

* <a id="脏象/五脏/肾">肾</a>：
    * 生理与病理：
        * 肾主藏精：
            * 藏本脏之精——先天之精：生殖之精。
            * 藏五脏六腑之精——后天之精：水谷化生之精。
            * 先天之精化生后天之精，后天之精供养先天之精。
            * 精的主要功能：
                * 主生殖。
                * 主生长发育。
                * 抵御外邪（提高免疫力）。
        * 肾主水液（肾的气化作用）：
            * 由水化气：将津液输布到全身，润养脏腑组织。
            * 由气化水：将代谢产物排出体外。
            * 水液代谢与肺、脾胃、肾、小肠大肠、膀胱有关。其中：
                * 肺为标，肾为本，脾为中流砥柱，肾的气化作用贯穿始终，居于重要地位。
                * 小肠主液、大肠主津：小肠能分清泌浊，上游是胃，
                下游是脾（清）和大肠（浊）；大肠还会对小肠产生的糟粕适当吸收一部分水分。
                注意，习惯上将小肠大肠这部分的功能归属于脾。
            * 详细的水液代谢过程详见[原始资料](#原始资料)。
        * 肾主纳气：“肾为气之根”，与肺遥相呼应，两者无虞方能呼吸顺畅。
        还要注意呼吸障碍“在肺为实，在肾为虚”，并且初病治肺，久病治肾。
        * 肾精骨髓可以化血：肾主骨，“骨者髓之府”，而骨髓是造血器官，
        骨髓又由肾精所生，所以肾精可以生血，临床上也常用补肾填精法治疗血虚证。
    * 与躯体官窍的关系：
        * 肾主骨、生髓、通于脑：肾藏精，精生髓，髓养骨，脑为髓之海。
        * 其华在发：
            * 肾精可以化血，头发要靠血液来濡养（“**发为血之余**”）。
            * 毛发的生机，根源于肾。头发的润泽与枯槁、生长与脱落，与肾精的盛衰有关。
            **肾虚或年老，头发易脱落、变白**。
        * 开窍于耳及二阴:
            * 两耳通脑，脑为髓之海。若肾精损耗，则髓海空虚，进而耳鸣耳聋。
            但耳又是宗脉之所聚，所以耳的病变与五脏皆有关，只有耳鸣耳聋的虚证，
            才责在肾亏。
            * 与前阴的关系有生殖、排尿（肾主水液）。
            * 与后阴的关系主要指肾与大便的关系，有二：
                * 肾阳不足：脾阳亦虚，大肠传化失职，可出现五更泻、虚寒下痢等证。
                * 肾阴亏损：不能滋润大肠，可导致便秘，常见于老年人和产妇。
    * 肾阴肾阳与命门学说：
        * 肾阴肾阳为各脏器的根本，均以肾精作为物质基础：
            * 肾阴：又称元阴、真阴、真水、肾水，是人体阴液的根本，对各个脏腑起着濡润、
            滋养的作用。肾阴虚不仅出现本脏阴虚证，还会引起他脏阴虚，反之亦然。
            * **肾阳**：又称元阳、真阳、**真火**、命门真火，是人体阳气的根本，
            对各个脏腑起着温煦、推动作用。肾阳虚不仅出现本脏阳虚证，
            还会引起他脏阳虚，反之亦然。注意某些理论的“火”是指肾阳而不是心火，
            例如引火归元。
            * “**五脏之阴气（阴液），非此不能滋；五脏之阳气，非此不能发**。”
        * 命门：
            * 位置有争议。
            * 但功能无争议，实际上就是肾阴和肾阳的作用，且典籍中更侧重与“阳”和“火”的作用，
            即命门之火（肾阳）。

### <a id="脏象/六腑">2.2 六腑</a>

* <a id="脏象/六腑/胆">胆</a>：
    * 贮藏和排泄胆汁：注意胆汁是在[肝](#脏象/五脏/肝)内形成。不过胆附于肝，肝胆本为一体，
    所以胆亦主疏泄（但以下降为顺），影响脾升胃降从而消化不良、纳运失职、厌食、腹胀等。
    若胆有病，多见胆火上逆而口苦、胁痛，若胆汁外溢则发为黄疸。
    * 胆主决断：
        * 正常的决断能力：即能够完全控制自己的意识和动作，与肝藏魂相似。
        * 准确，恰如其分，不偏不倚。
        * 若以上作用失调，则表现为病态的决断不能，或动作上的手不应心，准确失度，
        还影响人的勇怯，出现惊悸、虚怯、失眠等症。
    * 十一脏皆取决于胆：肝胆的疏泄与气血正常运行有密切关系，但因为胆为阳腑、肝为阴脏，
    由于阳统阴的关系，所以才有此说法。

* <a id="脏象/六腑/胃">胃</a>：
    * 胃主受纳：
        * 食物的第一站就是胃。
        * 若胃不受纳，则厌食或不思食。
    * 胃主腐熟：
        * 食物经过**初步**消化，成为**食糜**，**下行至小肠**（小肠部分功能归属到脾），
        其精微物质由脾运送至全身。
        * 若腐熟功能低下，则会胃脘胀满、疼痛、食积停滞等。
    * 胃气宜降：
        * 胃气降，食物才能向下传送到小肠，经过小肠分清泌浊后，浊者再下移到大肠变成粪便。
        * 若胃失通降，则食物及残渣就不能下行，就会出现胃脘胀满、疼痛、大便秘结等。
        * 若更甚者胃气上逆，则会出现嗳气、呃逆、恶心、呕吐等。
    * 喜润恶燥：
        * “阳明燥土，得阴自安”。
        * 若胃燥太过，津液不足，则水谷不能腐熟消化，会出现食经郁热、口舌干燥、
        口渴欲饮、大便秘结等症。
        * 在用苦寒泻下之剂时，必须在祛除实热燥结之后尽快停止，否则容易化燥伤胃阴。
    * 胃气的含义：
        * 一是泛指脾胃的消化功能：
            * 人以胃气为本，胃气在一定程度上代表了人的一般抗病能力。
            * 胃气可表现在食欲、舌苔、脉象和面色四个方面。一般将有食欲、舌苔正常、
            面色荣润、脉象从容和缓称之为有胃气。
        * 二是指脉的胃气：
            * 脉以胃气为本，一般将不浮不沉、不急不徐、从容和缓、
            节律一致的脉象称之为有胃气之脉。
        * 临床上往往以胃气之有无作为判断预后吉凶的重要依据，即“有胃气则生，
        无胃气则死”。

* <a id="脏象/六腑/小肠">小肠</a>：
    * 受盛：接受胃中的水谷。
    * 化物：进一步消化。
    * 分清泌浊：
        * 吸收清者，再通过脾的转输而至全身各部。
        * 浊者或下注大肠，形成大便；或渗入膀胱，形成小便。
        * 若失常，会出现腹胀、腹泻以及小便异常。

* <a id="脏象/六腑/大肠">大肠</a>：
    * 吸收水分：
        * 接受小肠下注的食物残渣，吸收其中剩余的水分，并将糟粕变为粪便。
        * 虚寒，就会无力吸收水分，则见肠鸣、腹痛、泄泻；
        * 实热，就会消烁水分过多，则见肠液干燥、大便秘结不通。
    * 排泄糟粕：
        * 将粪便由肛门排出。
        * 功能减弱，则便秘；
        * 功能亢进，则泄泻。

* <a id="脏象/六腑/膀胱">膀胱</a>：
    * 贮尿、排尿：也是主持人体水液代谢的器官，主要靠肾的气化（排尿）和固摄（贮尿）作用，
    合称肾之开合作用。若出现排尿异常，除了本腑直接感受湿热外邪等原因外，多由肾病所累及。

* <a id="脏象/六腑/三焦">三焦</a>：有争议，暂且理解如下：
    * 上焦：头、胸、心、肺。
    * 中焦：膈下脐上的腹部、脾、胃。
    * 下焦：脐下的腹部、阴部、肝、肾、膀胱、小肠、大肠。

### <a id="脏象/奇恒之腑">2.3 奇恒之腑</a>

* 名称的由来：形态中空，属腑，但具有藏精功能（一般的腑主传化而不藏），与脏类似，
故得此名。

* 明细项：包括脑、髓、骨、脉、女子胞（子宫），以及前述的[胆](#脏象/六腑/三焦)。
描述均很简短，详见[原始资料](#原始资料)原文。

### <a id="脏象/脏腑之间的关系">2.4 脏腑之间的关系</a>

* 脏与脏之间的关系：
    * [心](#脏象/五脏/心)与[肺](#脏象/五脏/肺)：
        * 生理：气与血的关系：“气为血之帅，血为气之母”。
        * 病理：
            * 肺气虚弱，导致心血阻滞：心悸、心胸闷痛、气短、唇青舌紫等。
            * 心气、心阳不足，导致肺失宣肃：喘咳、胸闷憋气、咳吐血痰等，
            最终心肺两气虚、血淤。
            * 温邪犯肺，逆传心包：导致卫气营血功能的失常。
    * [心](#脏象/五脏/心)与[脾](#脏象/五脏/脾)：
        * 生理：表现在血液的生成和运行方面：
            * 生成：心之血要靠脾的化生，脾的运转又要靠心行血来濡养。
            * 运行：血液的运行，既依赖心气的推动，又靠脾气的统摄
            （方能循经运行而不溢于脉外）。
        * 病理：
            * 脾病及心：脾虚统摄无权，可致心血亏耗，出现心悸、失眠或出血等症。
            * 心病及脾：心血不足，可致脾失所养，出现食欲不振、腹胀便溏、四肢倦怠等。
            * 以上结局都是心脾两虚，只不过是孰先孰后，何者为因、何者为果而已，
            解决办法均是用补益心脾之剂以治之。
    * [心](#脏象/五脏/心)与[肝](#脏象/五脏/肝)：
        * 生理：主要表现在血液和精神情志方面：
            * 血液：“肝藏血，心行之”。前者能濡养肝体，作为疏泄的物质基础，
            疏泄正常则与心行血相配合，使气血疏通，从而使肝有所藏、心血充盈。
            * 精神情志：心主神志，肝主疏泄，血液是神志活动的物质基础。
        * 病理：
            * 心血不足症候：心悸、失眠、面色不华等。
            * 肝血不足症候：头晕目眩、月经减少、爪甲不荣或干枯变形、手足麻木、肢体拘挛等。
            * 肝血不足导致心血亏损：肝血不足症候，兼心血不足症候。
            * 心血不足导致肝血亏虚：心血不足症候，兼肝血不足症候。
            * 心肝血虚导致精神情志异常：心烦失眠、精神抑郁、烦躁易怒等。
            * 心肝火旺而致精神异常：精神抑郁、神志痴呆、语无伦次、哭笑无常等。
    * [心](#脏象/五脏/心)与[肾](#脏象/五脏/肾)：
        * 生理：
            * 心肾相交：水火相济，则心阳不亢，肾水不寒。
            * 心为君火，肾为相火（命门火）：前者在上，为一身之主宰；后者在下，
            为阳气之根、神明之基础。
            * 精血互生。
            * 心藏神、肾藏精：精是神的物质基础，神是精的外在表现。心肾均与精神思维活动有关。
        * 病理：
            * 心肾不交：肾阴虚于下、心阳亢于上，而见心悸、怔忡、失眠、多梦、腰膝酸软、
            遗精等。
            * 水气凌心：心肾阳虚，寒水不化而上凌于心，症见心悸、怔忡、水肿，甚者喘不得卧等。
            * 阴虚火旺：心火上炎，出现口舌生疮、口干少津、五心烦热、舌红、脉细数等。
            * 精亏血少：肾精亏损和心血不足常互为因果，可同时出现心血不足和肾虚的证候。
            * 精神思维失常：心血亏损，肾精不足，均可见失眠、健忘、头晕、多梦等。
    * [肺](#脏象/五脏/肺)与[脾](#脏象/五脏/脾)：
        * 生理：主要表现为气和水方面：
            * 肺主气，脾益气：脾主运化，为气血生化之源，供应肺所需的津气；脾所化生的水谷之气，
            又依赖肺气的宣降方能敷布全身。
            * 肺为贮痰之器，脾为生痰之源：肺主通调水道，脾主运化水湿，两者互相合作，
            才能使体内的水液代谢正常运行。
        * 病理：
            * 气：肺虚累脾，脾虚及肺，形成恶性循环，常出现食少、便溏、消瘦、面色苍白、懒言、
            咳嗽等脾肺俱虚的证候。治疗时，在治肺的同时，也经常同时健脾。
            * 水液代谢：代谢障碍则水湿不化聚为痰饮，出现久咳不愈、痰多而稀白之候，
            病象多表现在肺而病本却在脾，这种痰饮咳嗽的治疗通常以健脾燥湿与肃肺化痰同用。
    * [肺](#脏象/五脏/肺)与[肝](#脏象/五脏/肝)：
        * 生理：主要表现在气血的升降运行上，肺主肃降，肝主升发。
        * 病理：
            * 肝火犯肺：又称木火刑金，病因是肝气郁结，气郁化火，向上灼肺伤津，
            影响肺之宣肃，症见咳嗽咽干、咳引胁痛，甚或咯血等。
            * 肺吸肝阴：肺失清肃，向下灼伤肝肾之阴，则肝使调达，疏泄不利，
            则在干咳无痰、咽喉干燥的同时，还伴有胸满胁痛等症，甚至肝阳偏亢而虚风内动。
    * [肺](#脏象/五脏/肺)与[肾](#脏象/五脏/肾)：
        * 生理：主要表现在水液代谢和呼吸方面：
            * 肺为水之上源，肾为主水之脏：水液只有经过肺气的宣发和肃降，
            才能到达全身各器官组织，并下输膀胱，而肾阳则为人体诸阳之本，
            其气化作用有升降水液的功能。
            * 肺为气之主，肾为气之根：肺司呼吸，肾主纳气。
        * 病理：
            * 水液代谢障碍：治病的关键是以肾为本，以肺为标，以脾为中流砥柱。
            * 肾不纳气：症见气短喘促、呼多吸少、动则尤甚，须用补肾纳气之法。
            * 肺肾阴虚：肺肾阴液的互相滋养称为金水相生，一方有损会涉及另一方，
            会出现腰酸膝软、潮热、盗汗、咽干、颧红、干咳 、音哑、男子遗精、女子闭经等证。
    * [肝](#脏象/五脏/肝)与[脾](#脏象/五脏/脾)：
        * 生理：主要表现在消化和血液方面：
            * 消化：肝主疏泄，分泌胆汁，助脾消化；脾主运化，为气血生化之源，润养肝体。
            * 血液：脾能生血统血，则肝有所藏；肝血充足，则疏泄正常，反过来增强脾之消化吸收。
        * 病理：
            * 消化：
                * 肝病及脾（胃）：又称肝木乘脾、木郁乘土、木郁土中，是肝气郁结、疏泄失职、
                肝气横逆克脾犯胃，导致脾胃虚弱、升降失常，症状除了有情绪抑郁、两胁胀满的肝症之外，
                还兼有腹胀肠鸣、大便溏泻等脾症（称为“肝脾不和”），或兼有胃脘疼痛、嗳气吞酸、
                恶心呕吐等胃症（称为“肝胃不和”）。
                * 脾病及肝：又称土壅侮木，可起因于湿遏脾阳，湿郁蕴热，致使肝胆疏泄不利，
                胆汁外溢而发为黄疸；或者脾虚被肝木乘侮而生风（称为脾虚生风，又叫慢脾风）
            * 血液：脾虚运行无力，或脾不统血、失血过多，均会累及于肝，造成肝血不足，
            出现食少、消瘦、眩晕、视物模糊、肢麻、经少甚至闭经、咳血、衄血等症。
    * **[肝](#脏象/五脏/肝)与[肾](#脏象/五脏/肾)**：
        * 生理：
            * **肝肾同源**：又叫精血同源、乙癸同源，两脏**同盛同衰**，主要表现如下：
                * 肝肾相生：肾阴滋养肝阴，使其能制约肝阳不致过亢，维持正常的疏泄功能；
                而肾阴的再生，又须通过肝的疏泄而入藏于肾。所以说，肝肾阴液是互相资生的关系。
                * 精血同源：肝藏血，肾藏精。肾精滋养于肝，使肝血充足，维持正常的疏泄功能；
                反过来，肾精又赖肝血的不断补充而化生，维持肾阴与肾阳的动态平衡。可见，
                肝肾均需精血濡养，即精血同源。
                * 肝肾同具相火：相火是与心之君火相对而言。一般认为肝、肾、胆、三焦均内具相火，
                而相火又源于命门，所以同源。
            * 事关生殖：肝藏血，肾藏精，精血是男子排精和女子月经的物质基础，且肝主筋，
            前阴又是宗筋之所聚，所以肝肾均与生殖有密切关系。
        * 病理：
            * 肝肾虚证：
                * 肾阴（精）亏耗，可导致肝阴不足、肝阳偏亢，称为水不涵木；
                * 肝阴（血）不足，亦可导致肾阴不足。
                * 上述两种现象会互为因果，最终导致肝肾阴虚，出现：
                    * 肾阴虚于下的症状：腰酸膝软、遗精、滑精、五心烦热等。
                    * 肝阳亢于上的症状：眩晕、头痛、易怒、耳鸣耳聋等，
                    甚至引起虚风内动而手足蠕动或抽搐。
                * 治疗时宜**补肾**与养肝同用。此处补肾就是补肝，例如杞菊地黄丸，
                就是通过滋补肾阴而滋上亢之肝阳。
            * 肝之实证：肝火、肝阳亢盛，不但能耗伤肝阴肝血，也可损伤肾精，下吸肾阴，
            导致肾阴虚损、肾阳偏亢而相火妄动，出现：
                * 肝阳亢盛的症状：眩晕、头痛、耳鸣、失眠、面红目赤等。
                * 肾阴不足的症状：腰酸膝软、梦交、遗精、阳事易举等。
                * 治疗时宜使用滋肾**泻肝**法。此处泻肝就是泻肾火，例如六味地黄丸，
                就用[丹皮](#牡丹皮)来清泻肝火；知柏地黄丸，则是用[知母](#知母)、
                [黄柏](#黄柏)苦寒坚阴之品，平相火而保真阴，是壮水与制火并举。
            * 疏泄与闭藏关系失调：会导致女性月经紊乱或闭经，男性遗精、滑精、梦交等。
    * [脾](#脏象/五脏/脾)与[肾](#脏象/五脏/肾)：
        * 生理：
            * 先天与后天互相资生：脾必须得到肾阳的温煦蒸化，才能健运；
            肾精必须依靠脾阳运化而得的水谷精微，才能充盈。
            * 水液代谢：肾主水液，脾主运化水湿（土可制水），又在中央而起转输作用，
            所以两者功能正常，就能协同其他脏腑，共同完成水液代谢。
        * 病理：
            * 消化机能障碍：脾失健运，则水谷不化；肾阳不足，阴寒极盛，则胃关不固。
            两者皆可生泄泻，出现腹部冷痛、下利清谷、五更泻（又叫脾肾泻）等症。
            * 水液代谢紊乱：脾虚不能制水，肾虚不能主水，脾阳虚和肾阳虚也互相影响，
            陷入恶性循环，最终引起水液停聚，发生浮肿或腹水等。

* 腑与腑之间的关系：
    * 化水谷、行津液，分工合作。
    * 不断地受纳排空、虚实更替、实而不满，故有“六腑以通为用”、
    “腑病以通为补”的说法。

* 脏与腑之间的关系：
    * [心](#脏象/五脏/心)与[小肠](#脏象/六腑/小肠)：
        * 生理：相为表里，两者之间有经络相连。
        * 病理：
            * 心移热于小肠：除了心的热症，还会有小便短赤、尿道灼热疼痛甚至尿血等小肠实热之症。
            * 小肠实热上熏于心：除了小肠的热证，还会有心烦、舌尖红赤疼痛、
            口腔糜烂等心火炽盛的症状。
    * [肺](#脏象/五脏/肺)与[大肠](#脏象/六腑/大肠)：
        * 生理：肺气肃降，大肠之气亦随之而降，发挥其传导功能；大肠传导通畅，
        则肺气才能和利。此外，肺和大肠亦是人体水液代谢过程的两个环节。
        * 病理：
            * 肺失肃降导致大肠传导受阻：
                * 实热便秘：首先采用泻热通便之剂，效果不佳再加泻肺、肃肺、宣肺之品。
                * 气虚便秘：宜补益肺气、润肺通便。
            * 大肠实热导致肺气不降：
                * 肺气上壅、阳明腑实：前者可见喘促、胸满，后者可见高热、神昏谵语、
                腹满硬痛拒按、大便热结不通等，可采用通里攻下、釜底抽薪之法，
                泻大肠以降肺气。
    * **[脾](#脏象/五脏/脾)与[胃](#脏象/六腑/胃)**：
        * 生理：
            * 脾主运化，胃主受纳：胃的受纳和腐熟水谷，是为脾的运化作准备；
            脾的运化，是为胃的继续受纳和腐熟提供能源。
            * 脾主升清，胃主降浊：脾的升清作用，能化生气血将其上输到心肺以营养全身；
            胃的降浊作用，能将食物残渣下传到小肠作进一步消化和排泄。
            * 脾喜燥恶湿，胃喜润恶燥：脾为阴脏，以阳气用事，脾阳健则能运化；
            胃为阳腑，需阴液滋养，阴足则能受纳和腐熟。
        * 病理：
            * 纳运失常：
                * 主病在脾：消化不良、食后饱胀、大便稀薄，是脾不健运，需健脾醒脾。
                * 主病在胃：食欲不振或嘈杂易饥，是胃不受纳或胃太亢进，需开胃和胃。
                * 由于脾胃互相影响，所以以上脾胃症状往往同时并见，两种治疗方法也常常同用。
            * 升降反作：
                * 脾气不升：消化不良、食后腹胀、腹泻便溏，或眩晕乏力、气短懒言、面色萎黄等，
                甚则脾气下陷（亦称中气下陷）而出现脱肛、子宫脱垂和内脏下垂等证候。
                治疗上宜用升药，如[升麻](#升麻)、[葛根](#葛根)之类。
                * 胃气不降：食欲不振、胃脘胀满、嗳气、恶心、呕吐等。治疗上宜用降药，
                如[黄连](#黄连)、[大黄](#大黄)之类。
                * 清气不升必然导致浊气不降，反之亦然。“清气在下。则生飧泄；浊气在上，则生䐜胀”。
                在实际中，脾气不升和胃气不降往往同现。
            * 燥湿失度：
                * 脾湿：外湿袭脾，或脾失健运而生湿，均会引起水湿停滞为患，
                可采用苦温燥湿法（李东垣倡导“升阳除湿”），达到健脾的目的。
                * 胃燥：热邪犯胃灼伤胃阴，或胃气上逆而频繁呕吐导致胃津耗损，均会出现燥象，
                多用润燥养胃法（叶天士首倡“养胃阴”），达到和胃的目的。
        * 关系总结对比表：略。
    * [肝](#脏象/五脏/肝)与[胆](#脏象/六腑/胆)：
        * 生理：
            * 两者的经脉互相络属而为表里。
            * 肝在胁下，胆附于肝。
            * 胆汁产生于肝，贮藏于胆，两者协作而共同疏泄胆汁到肠道帮助消化。
            * 肝的疏泄正常，胆才能贮藏到应有的胆汁；胆汁排泄无阻，肝才能正常疏泄。
        * 病理：
            * 在病理上互相影响，常常肝胆证候同时出现，例如：
                * 肝胆湿热而致的黄疸：既有胆汁外溢之发黄、口苦等症，又有胁痛、
                胁胀等肝气郁结的症状。
                * 肝火上炎引发胆气上逆：既有肝郁化火之症，又有口苦咽干的胆失疏泄之候。
            * 大部分疏肝理气的药物，也都有不同程度的利胆作用。
    * [肾](#脏象/五脏/肾)与[膀胱](#脏象/六腑/膀胱)：
        * 生理：
            * 肾气的固摄作用控制膀胱贮存尿液。
            * 肾气的气化作用控制膀胱排泄尿液。
        * 病理：
            * 膀胱虚证：通常因为肾阳虚衰、气化失司、开合不利而对膀胱失控，
            可见小便频数、余沥不禁甚或遗尿等症，治疗时以温肾固气为主。
            * 膀胱湿热：膀胱本身受湿热侵袭，或肾移热于膀胱，可出现小便短赤或癃闭、
            尿血等症，宜采用清热利湿之法进行治疗。

### <a id="脏象/气、血、津液、精、神">2.5 气、血、津液、精、神</a>

* <a id="脏象/气、血、津液、精、神/气">气</a>：
    * 基本概念：根据不同的语境，可有如下含义：
        * 流动着的细微物质：如呼吸之气、水谷之气等。
        * 脏腑组织的生理功能：如脾气可指脾的运化、生血统血等功能。
        * 脏腑组织的病理特点：例如肺气不宣、肺气上逆等。
    * 生成：
        * 肾中之精气：形成人体的原始物质，称为先天之气。
        * 水谷之精气：饮食经脾胃的消化、吸收而来的水谷精微，是后天之气。
        * 吸入的清气：自然界的新鲜空气，又称天气。
        * 肺、脾、肾的功能：尤以脾胃最为重要，即“气之源头在乎脾”，
        所以补气主要是补脾肺两脏之气。
    * 运行形式：升、降、出、入，原书介绍得比较简短，
    详细规律在[后面小节](#脏象/气化概论/基本规律/气化运动与脏腑的关系)：
    * 生理功能：
        * 推动作用：促进人体的生长发育、执行各脏腑组织的生理活动、行血、输布津液。
        * 温煦作用：维持正常体温。
        * 防御作用：护卫肌表，抵御外邪。
        * 固摄作用：控制血液不溢于脉外、规律排汗/尿/精。
        * 气化作用：
            * 一是泛指物质和能量的新陈代谢，即气、血、津液、精之间的互相化生。
            * 二是指脏腑的某些功能活动，如肾对水液的调节称为肾的气化，
            膀胱对尿液的调节称为膀胱的气化。
    * 分类：
        * 元气（原气、真气、真元之气、生气、先天之气）：
            * 生成：发源于肾（命门），包括元阴元阳之气（即肾阴肾阳之气）。
            * 分布和运行：秘藏于肾精之中，通过三焦，到达身体各个部分。
            * 功能：是生命活动的原动力、最基本的物质。
        * 宗气：
            * 生成：宗气= 自然界清气 + 脾吸收的水谷之气 = 营气 + 卫气
            * 分布和运行：下注气街、上走息道。
            * 功能：下行贯心脉而行气血，上行走息道而司呼吸，是“**肺气**”的**同义词**。
        * 营气：
            * 生成：脱胎于宗气、运于脉中的精气。
            * 分布和运行：
                * 十二经整体运行：暂略。
                * 任督循环：暂略。
            * 功能：化生血液，并随血运行以营养周身。
        * 卫气：
            * 生成：脱胎于宗气、附行于脉外的精气。
            * 分布和运行：
                * 昼行于阳`25`周次：暂略。
                * 夜行于阴`25`周次：暂略。
                * 行于**阳则寤**，行于**阴则寐**。
            * 功能：
                * 护卫肌表，防御外邪。
                * 司汗孔开合，调节体温。
                * 温煦脏腑，润泽皮毛。

* <a id="脏象/气、血、津液、精、神/血">血</a>：
    * 生成：
        * 水谷精微化而为血：注意脾的生血造血功能，以及脾藏营、营血并称的关联关系。
        * 肾精骨髓可化为血：肾精能生髓，骨髓可以生血。“气不耗，归气于肾而为精；
        精不泄，归精于肝而化清血”。
        * 肝藏血而又生血：藏血显易而见；至于生血，则由于肝肾同源，“归精于肝而化清血”。
    * 循行：
        * 心主血脉：心气提供基本动力。
        * 肺朝百脉：肺气助心行血，并将血液敷布全身。
        * 肝主藏血：肝之疏泄和藏血功能可以起到调节血量的作用，即静时归入于肝、
        动时奔流各处。
        * 脾主统血：脾气能统摄、控制血液不溢于血管之外。
    * 生理功能：
        * “血主濡之”，能濡润滋养全身，供给营养。
        * 也是精神活动的主要物质基础。

* <a id="脏象/气、血、津液、精、神/津液">津液</a>：
    * 基本概念：
        * 是人体内除了血液之外的正常水液的总称，主要指体液，也包括一些分泌物，
        如唾液、胃液、肠液、关节腔内液体，以及泪、涕、汗、尿等。
        * 津：体液中清轻稀薄的部分，渗透在皮肤、肌肉之中，能充养皮肤、滋润肌肉，
        流动性较大。
        * 液：体液中重浊黏稠的部分，充盈在关节、脑髓和孔窍等处，能润滑关节、
        补益脑髓和濡养孔窍，流动性较小。
        * 津液均由水谷精微化生而来，且两者互相影响、转化。在病理上，伤津能伤液，
        脱液也能伤津，故常常津液并称，不予严格区别。
    * 代谢：依赖**肺、脾、肾**、大小肠、膀胱、三焦等脏腑的共同作用，详细过程略。
    * 生理功能：
        * 津：
            * 营养和滋润脏腑、肌肉、经脉、皮肤。
            * 生成血液，不断补充血液中的水分。
        * 液：
            * 填精补髓。
            * 滑利关节，濡养空窍。
        * 调节体温、阴阳平衡。

* <a id="脏象/气、血、津液、精、神/精">精</a>：
    * 基本概念：见前面“[肾](#脏象/五脏/肾)主藏精”一节，以及[原始资料](#原始资料)原文。
    * 分类：同上。
    * 生理功能：同上。

* <a id="脏象/气、血、津液、精、神/神">神</a>：
    * 基本概念：见前面“[心](#脏象/五脏/心)藏神”一节，以及[原始资料](#原始资料)原文。
    * 生成：同上。
    * 生理功能：同上。

* 气、血、津液、精、神之间的关系：
    * [气](#脏象/气、血、津液、精、神/气)和[血](#脏象/气、血、津液、精、神/血)的关系：
        * 气能生血：注意此处的气，概念已经非常宽泛，既指生命活动所需要的一切物质
        （包括精、血、津液），也指气化作用和各个脏腑功能，理解这些才能理解为什么说气生血，
        而通常不说血生气（但气也需要精、血、津液提供的水谷精微来不断补充）。
        * 气能行血：气为血之帅，心气推动、肺气助心行血、肝调节血量、脾统摄血液。
        * 气能摄血：主要是脾气的作用。
        * 血为气母（此处的气为狭义的气）：
            * 气必须依附于血，否则气将飘浮不定，无以所归。
            * 气需要精、血、津液提供的水谷精微来不断补充。
        * 小结：
            * 气属阳，血属阴。
            * 血无气不行，气非血不载。
            * 气即无形之血，血即有形之气。
            * 气为主、为重，血为辅、为轻，血有不足可以渐生，若气不立即死矣。
    * [气](#脏象/气、血、津液、精、神/气)和[津液](#脏象/气、血、津液、精、神/津液)的关系：
        * 气可化水，水停气阻：津液的生成、输布和排泄，均靠气机的升降出入运动，
        离不开肺、脾、肾、三焦、膀胱等脏腑的气化功能。若气化失司，则津液新陈代谢受阻，
        出现水液潴留、痰饮停聚，反过来又会影响脏腑气机的升降出入。
        * 水可化气，气随液脱：水谷化生的津液，从中焦进入肺脉，化而为气、血、精等，
        而且气必须依附于津液而存在，否则将漂浮不定、无以所归。因此，当津液散失，
        也必导致气的耗损，如大汗可亡阳、剧吐之下无完气、暑病可伤津耗气等。
    * [气](#脏象/气、血、津液、精、神/气)和[精](#脏象/气、血、津液、精、神/精)的关系：
        * 气归精：气可生精，又能固（摄）精。此处的气，主要指流动的细微物质，以及各脏腑功能。
        * 精化为气：精藏于肾，在肾阳的蒸动下，化为**元气**，通过三焦，升腾于上，布达周身。
    * [气](#脏象/气、血、津液、精、神/气)和[神](#脏象/气、血、津液、精、神/神)的关系：
        * 气帅血以养神。
        * 神为气之主而御气之动。
    * [血](#脏象/气、血、津液、精、神/血)和[津液](#脏象/气、血、津液、精、神/津液)的关系：
        * 血溢于脉外（包括皮肤外）则为化津液。
        * 津液为血液的组成部分，不断补充血液。
    * [血](#脏象/气、血、津液、精、神/血)和[精](#脏象/气、血、津液、精、神/精)的关系：
        * 血能化精：所以血液亏虚，每致肾精不足。治疗肾虚精衰者，每于填精药中兼以养血之品。
        * 精能生血：发为血之余，肾其华在发，肾精不足可致血虚而终成精血不足，
        其证不仅有血虚之征，亦有毛发枯槁甚至脱落之候。所以，治疗血虚亦可用补肾填精的药方。
    * [血](#脏象/气、血、津液、精、神/血)和[神](#脏象/气、血、津液、精、神/神)的关系：
        * 血是神的物质基础：血盛则神旺，血虚则神怯，血尽则神亡。
        * 神是血的主宰：血液的生成和运行均赖神以主宰。
    * [精](#脏象/气、血、津液、精、神/精)和[津液](#脏象/气、血、津液、精、神/津液)的关系：
        * 肾精足则肾的气化旺盛，津液的输布正常。
        * 津液又是肾精的组成部分，补精药多数也具有滋养津液的作用。
    * [精](#脏象/气、血、津液、精、神/精)和[神](#脏象/气、血、津液、精、神/神)的关系：
        * 精以化神：神成于先天之精，又赖后天水谷之精不断充养。精盛则神旺，精衰则神扰。
        * 神以化精：神为精之主宰，神守则志安而精固，神散则志乱而精失。
    * [津液](#脏象/气、血、津液、精、神/津液)和[神](#脏象/气、血、津液、精、神/神)的关系：
    津液是血液的组成部分，所以津液和神的关系与血和神的关系类似。
    * 气、血、津液、精、神之间的关系总结图：略。

### <a id="脏象/气化概论">2.6 气化概论</a>

* 基本概念：
    * 广义：人体内气的运动变化、能量转换，如食物的消化、营养的吸收、血液的运行、
    精气的转输、糟粕的排泄。
    * 狭义：专指三焦气化，即三焦输布**水液**的功能。

* 基本特征：
    * 概括起来就是：**新陈代谢**，即气、血、精、津液等物质的生成、转化、
    利用和排泄过程。
    * 气、精、形、味的关系：
        * “味归形，形归气，气归精，精归化”。此处的“气”可以理解为脏腑组织的生理功能。
        * “精食气，形食味”。此处的“食”可直接使用其原义，“气”主要指流动着的细微物质。
        * “化生精，气生形……精化为气”。此处的“气”理解成流动着的细微物质或者脏腑组织的生理功能均可。
        * “天食人以五气，地食人以五味”。此处的“食”通“饲”。

* 基本形式（又称为气机）：升、降、出、入。

* 基本规律：
    * <a id="脏象/气化概论/基本规律/气化运动与脏腑的关系">气化运动与脏腑的关系</a>：
        * 一般规律：
            * 升其清阳，降其浊阴，摄其所需，排其所弃。
            * 五脏贮藏精气，宜升。但心、肺在上，升（主要）中有降（次要）。
            * 六腑传导化物，宜降。但也有吸收精微、津液的作用，降（主要）中有升（次要）。
            * 升已而降，降已而升，升降中复有升降。
        * 具体形式：
            * 心：心血上荣头面、供养神明为升，下行营养周身为降，其位在上而以**降为主**。
            * 肺：肺气的宣发为升，肃降为降，其位最高而以**降为主**。
            * 肝：肝调节血量，疏泄于心脉，升发而上；又能向下疏泄精关，调节男精女血，
            还与肾配合而通调水道。由于肝为少阴，其位又在下，所以**主升发**，兼能降泄。
            * 肾：肾藏精，精可生血输布全身，也可生髓上充以养脑，还可产生男精女血向下排出。
            肾主水液，通过升清（上归于肺）降浊（下输膀胱）而调节水液代谢。因此，
            肾**亦升亦降**。
            * 脾：脾主升清，将水谷精微上输于肺而达全身，又将代谢产物下达于肾和膀胱，
            其位居中还能交通心肾，所以应重点关注其**枢纽**作用。
            * 脏腑之间的气机升降：
                * 肝升肺降：人身精气输布流行以肝肺为枢转，肝升使之上达头身及上窍，
                肝降使之下达脏腑及筋骨。
                * 心肾相交：心火下降使肾水不寒，肾水上济使心阳不亢。心主血，肾藏精，
                精血又能互生。心藏神，肾藏精，精与神又能互化。心肾相交则水火既济、
                阴阳平衡、精神安宁。
                * 脾升胃降：脾胃中土为脏腑气机升降的枢纽、轴心。
                * 侧重与平衡的艺术：从局部看，并不是每一种生理活动，都必须升降出入齐备，
                而是各有侧重；从整体看，升和降、出和入之间必须协调平衡，
                不及或太过都是病态。
    * 气化运动与经络的关系：循环规律与脏腑气机升降规律基本上是一致的，细节暂略。

* 呼吸、循环、消化、水液的气化运动：
    * 呼吸系统的气化运动：
        * 肺主呼气，肾主纳气。肾摄纳肺吸入的清气，转而为用，产生的浊气再上复于肺排出。
        * 肝主疏泄，脾能益气，所以肝脾亦与呼吸有关。
    * 循环系统的气化运动：主要与[心](#脏象/五脏/心)、[肺](#脏象/五脏/肺)、
    [肝](#脏象/五脏/肝)、[脾](#脏象/五脏/脾)有关，具体可查阅原书该小节的内容，
    或相关脏腑与血液循环相关的描述。
    * 消化系统的气化运动：
        * 饮食首先入**胃**，进行初步的腐熟和消化；
        * 再经过**脾**的运化、**小肠**的分清泌浊之后：
            * 清者（营养物质）由脾气上输于**肺**，与吸入的清气结合，然后贯注**心脉**，
            在肺气和心气的作用下，由**经脉**送到全身；
            * 浊者由胃气下降再传入小肠，进行二次“泌别清浊”，将糟粕（浊中之浊）下传大肠；
        * **大肠**吸收糟粕的部分水分，最后产生粪便排出体外。
        * 脾胃的腐熟和运化，还必须以**肝**之疏泄和**肾**阳的温煦为动力。
    * 水液系统的气化运动：
        * **肾**为本，**肺**为标，**脾**为中流砥柱，**肝**主疏泄，
        **三焦**为水液升降的道路。水液有清有浊，清浊交加，清者上升，浊者下降。
        * 肺中之水为清：
            * 清之清者：经肺气宣发，润养肌腠、皮毛、四肢、百骸，代谢废物为汗液。
            * 清之浊者：下输于肾。
        * 归肾之水为浊：
            * 浊之清者：复升于肺，重新参与代谢。
            * 浊之浊者：经由**膀胱**变成尿液。

## <a id="常用药提要">10、常用药提要</a>

### <a id="常用药提要/发散药">10.1 发散药</a>

* <a id="防风">防风</a>：
    * 性味归经：辛、甘，性微温，入脾、肝、膀胱经，偏入气分（[荆芥](#荆芥)偏入血分）。
    * 功效配伍：
        * 最常用的辛温发汗药：配[荆芥](#荆芥)（常同用）、[紫苏](#紫苏)叶，
        可治疗感冒风寒的表证。
        * 祛除经络筋骨中的风湿：常与[羌活](#羌活)、[独活](#独活)、[当归](#当归)、
        [薏米](#薏米)、[威灵仙](#威灵仙)、[伸筋草](#伸筋草)、[鸡血藤](#鸡血藤)等同用。
        * 祛风解痉：可用于治疗肝风内动、风痰上扰、破伤风等，需要与[全蝎](#全蝎)同用，
        能增强[全蝎](#全蝎)的作用，还可随证配用[钩藤](#钩藤)、[蜈蚣](#蜈蚣)、
        [僵蚕](#僵蚕)等。
        * 治疗因肝郁伤脾而致的腹痛、腹泻：常配合[白术](#白术)、
        [白芍](#白芍)等同用，典型处方是痛泻要方。
        * 治疗肠风便血：尤其是反复发作、日久不愈的大便下血，可配合[地榆](#地榆)炭、
        [槐角](#槐角)炭、炒[槐花](#槐花)等同用。
        * 杀附子毒：与[附子](#附子)同用可减小其毒性。
        * 增强[黄芪](#黄芪)的功效。
    * 份量用法：一般为`6`～`9`g，有时可用到`12`g。
    * 禁忌：
        * 本品恶[干姜](#生姜)、[藜芦](#藜芦)、[白蔹](#白蔹)、[芫花](#芫花)，
        畏[萆薜](#萆薜)。
        * 阴虚火旺、血虚发痉者慎用。
    * 造假鉴别：通常用于冒充[党参](#党参)，但防风苦，党参甜，可以此鉴别。
    * 熏硫鉴别：待补充。

* <a id="生姜">生姜</a>：
    * 性味归经：辛，微温，入肺、胃、脾经。
    * 功效配伍：
        * 发汗解表：可配[麻黄](#麻黄)、[荆芥](#荆芥)、[桂枝](#桂枝)、
        [紫苏](#紫苏)叶等同用，也可单用生姜一味，加适量红糖煎汤饮服，
        治风**寒**感冒。
        * 和胃止呕：一般可配[陈皮](#陈皮)，若是寒湿痰饮可配[半夏](#半夏)，
        若属热证呕吐则配[竹茹](#竹茹)、[黄连](#黄连)。
        * 益脾胃元气，温中祛湿：配[黑枣](#黑枣)，寒重则加[砂仁](#砂仁)、
        [白术](#白术)、[半夏](#半夏)。
        * 制[白芍](#白芍)之寒。
        * 温肺、化痰、止咳：可与[桂枝](#桂枝)、[细辛](#细辛)、[款冬花](#款冬花)等同用。
        * 解半夏、天南星及鱼蟹鸟兽肉毒。
        * 另有生姜皮，以及炮制方法不同的干姜、煨姜、炮姜，
        功效作用见[原始资料](#原始资料)。
    * 份量用法：`3`～`10`g。通常不写入主处方，也不从药店购买，
    而是单独购买。
    * 禁忌：用量不宜太大，否则**可能**刺激肾脏发炎。阴虚内热者少服或忌服。

* <a id="菊花">菊花</a>：
    * 性味归经：甘、苦，性微寒，入肺、肝经。
    * 功效配伍：常用于治疗风热感冒或头痛，也是**眼科常用药**，作用如下：
        * 疏散风热：可用于风热感冒，常配合[桑叶](#桑叶)、[薄荷](#薄荷)、
        [荆芥](#荆芥)、[防风](#防风)等同用。
        * 平肝明目：
            * 对于风热上攻、目赤肿痛，可与[密蒙花](#密蒙花)、[桑叶](#桑叶)、
            [蝉蜕](#蝉蜕)等同用；
            * 对于肝阴不足、虚火上炎，可与[地黄](#生地)、[白芍](#白芍)、[蔓荆子](#蔓荆子)、
            [枸杞子](#枸杞子)等同用。
        * 清热解毒：尤善解疔毒（初起时），要用野菊，内服外用皆可，
        常配合[金银花](#金银花)、[连翘](#连翘)、[蒲公英](#蒲公英)、
        [紫花地丁](#紫花地丁)等同用。
    * 份量用法：
        * 一般为`6`～`9`g，特殊情况也可用`12`～`15`g。
        * 入汤剂时最好**后下**，煎`5`～`10`分钟。
        * 疏散风热多用黄菊，平肝明目多用白菊，清热解毒多用野菊。可长服。
    * 禁忌：凡阳虚或头痛而恶寒者忌用。

* <a id="柴胡">**柴胡**</a>：
    * 性味归经：苦，性平偏寒，入肝、胆、心包络。
    * 功效配伍：
        * 和解**少阳**：症见寒热往来、胸胁苦满、口苦咽干、食欲不振、心烦欲呕时，
        可配[黄芩](#黄芩)、[党参](#党参)、[甘草](#甘草)、[生姜](#生姜)、
        [黑枣](#黑枣)等同用，并根据是否有口渴、咳嗽等症状进行药物加减，
        具体不在此展开。
        * **疏肝解郁**：常配合[当归](#当归)、[白芍](#白芍)、[白术](#白术)、[茯苓](#茯苓)、
        [薄荷](#薄荷)、[香附](#香附)、[川芎](#川芎)、[枳壳](#枳壳)、[元胡](#元胡)等同用。
        典型处方有逍遥散、柴胡疏肝散等。
        * **升举阳气**：可治因脾胃虚弱、清阳下陷所致的气短腹坠、肛门下坠、小便频数、脏器下垂、
        子宫脱垂等症，常配合炙[黄芪](#黄芪)、[白术](#白术)、**[升麻](#升麻)**、[防风](#防风)、
        [葛根](#葛根)、[党参](#党参)等同用。典型处方有补中益气汤、升阳益胃汤、
        升提汤等。
        * **治疟疾**：疟疾多有寒热往来之症，所以治法也与和解少阳相似，随证加减。
    * 份量用法：用量一般为`0.9`～`9`g，重症有时可用至`30`g，退热及治疟时可用`10`～`15`g。
    用于升阳时，用量宜小。处方上只写柴胡时，药房默认给北柴胡（药力最强，功效最全），
    另有南柴胡、银柴胡、竹叶柴胡不在此展开。
    * 禁忌：阴虚内热、阳气易升动（尤其是肝阳上亢）者忌用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="升麻">升麻</a>：
    * 性味归经：甘、辛，性微寒，入肺、脾、胃经。
    * 功效配伍：
        * 发表透疹：用于麻疹初期透发疹毒，常与[葛根](#葛根)同用增强发表之力。
        * 清热解毒：善解**阳明**热毒，用于胃热齿痛、口舌生疮、皮肤痒疹等，
        可配[石膏](#石膏)、[黄连](#黄连)等同用，典型处方有清胃散。
        * 升阳举陷：善升脾胃之阳气，是治疗**脏气下陷**的**必用之品**，常与[柴胡](#柴胡)、
        [人参](#人参)、[黄芪](#黄芪)同用。
    * 份量用法：`3`～`6`g。蜜制则升阳力增强，炒制则收敛止血。
    * 禁忌：吐血、鼻衄、咳嗽多痰、阴虚火动、上盛下虚及疹已透者忌用。

### <a id="常用药提要/泻利药">10.2 泻利药</a>

* <a id="茯苓">茯苓（含茯苓皮、茯神、茯神木）</a>：
    * 性味归经：甘、淡，性平，入心、脾、胃、肺、肾经。
    * 功效配伍：补而不峻，利而不猛，既能扶正，又能祛邪，临床多作辅助药，
    与补气、健脾、利水药同用，概括如下：
        * 利水渗湿、利尿消水：五脏六腑的水湿停留证候均可用本品，
        可配合[党参](#党参)、[白术](#白术)、[陈皮](#陈皮)、[猪苓](#猪苓)、
        [泽泻](#泽泻)、[桑白皮](#桑白皮)、[冬瓜皮](#冬瓜皮)等同用。
        * **宁心安神**：可治失眠健忘、心脾两虚、心神不宁之证，
        可配合[当归](#当归)、[白术](#白术)、[柏子仁](#柏子仁)、[远志](#远志)、
        （炒）[酸枣仁](#酸枣仁)等同用，且在安神方面，茯神比茯苓更好，
        并且两者往往同用。
        * 益脾止泄：配[党参](#党参)、[白术](#白术)、[猪苓](#猪苓)、[泽泻](#泽泻)、
        [藿香](#藿香)、[车前子](#车前子)、炒[芡实](#芡实)、[伏龙肝](#伏龙肝)等，
        可治由脾虚湿盛引起的水泻；配[党参](#党参)、[白术](#白术)、[甘草](#甘草)，
        可治脾虚气弱等证。
    * 份量用法：茯苓、茯神（抱松根而生者）一般为`9`～`12`g，茯苓皮、
    茯神木（茯神中的松根）可用`15`～`30`g。
    * 禁忌：
        * 忌米醋。
        * 阴虚津枯或小便过多者忌用。
        * 虚寒滑精或气虚下陷者慎用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

### <a id="常用药提要/补益药">10.3 补益药</a>

* <a id="人参">人参</a>：
    * 性味归经：甘、微苦，生者性平、熟则性温，入肺、脾经。
    * 功效配伍：补五脏，宁神益智，健脾补肺，益气生津，大补人体**元气**，常见配伍如下：
        * 抢救虚脱：典型处方有独参汤、参附汤、生脉散等。
        * 治疗气虚：常与[白术](#白术)、[茯苓](#茯苓)、[甘草](#甘草)、[黄芪](#黄芪)、
        [山药](#山药)、[五味子](#五味子)等同用，可补脾肺而治气虚。
        典型处方则有四君子汤、补肺汤、人参养荣汤、补中益气汤等。
        * 扶正祛邪：可在祛邪的药方中加入本品，增加扶正功效。
        * 宁神益智：用于心脾两脾，症见惊悸健忘、疲劳乏力、神经衰弱等，可与[黄芪](#黄芪)、
        [白术](#白术)、[酸枣仁](#酸枣仁)、[龙眼肉](#龙眼肉)、[茯苓](#茯苓)等同用，
        如人参归脾丸。
    * 份量用法：
        * 一般为`1.5`～`9`g，用于独参汤等救急时，可用`9`～`30`g。
        * 一般情况下可用[党参](#党参)代替，但抢救急症（如虚脱、休克等）及治重症时，
        则用本品为宜。
        * 市面售卖的人参可分为野生和人工种植的：
            * 野山参（老山参）：大补元气，无温燥之性，能滋养阴津，但货源少、
            价格贵，很少用。
            * 红人参（人工种植）：带刚健温燥之性，能振奋阳气，适用于急救回阳。
            * 生晒参（人工种植）：性较平和，不温不燥，既可补气，又能生津，
            适用于扶正祛邪。
            * 白人参（人工种植）：也叫糖参，性最平和，但效力也小，适用于健脾益肺。
            * 高丽参（人工种植）：产于朝鲜，也有红、白、生晒参之分，效力与用法同上。
    * 禁忌：
        * 反藜芦，畏五灵脂。
        * 凡肺气壅滞、胸闷憋气、表邪未解，以及一切实证、热证，均忌用。
        * 如服后腹胀太甚，可用[莱菔子](#莱菔子)或[山楂](#山楂)解之。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="党参">**党参**</a>：
    * 性味归经：甘，性平，入肺、脾经。
    * 功效配伍：常作为[人参](#人参)的代用品：
        * 健脾胃：可配[白术](#白术)、[茯苓](#茯苓)、[甘草](#甘草)、[陈皮](#陈皮)，
        即“五味异功散”；或配[白术](#白术)、[山药](#山药)、[扁豆](#扁豆)、[芡实](#芡实)、
        [莲子](#莲子)肉、[薏米](#薏米)、[茯苓](#茯苓)等，即“参苓白术散”。
        * 益气补血：常配合[当归](#当归)、[白芍](#白芍)、[生地黄](#生地)、
        [熟地黄](#熟地)等，治疗各种贫血，还可参考经典处方：八珍汤、人参养荣汤（党参代人参）、
        归脾汤。

        * 生用养阴、补血、柔肝，酒炒（减其寒性）和中缓急，土炒安脾止泻。
    * 禁忌：
        * 阳衰虚寒之证不宜单用。
        * 凡中寒腹痛腹泻、肠胃虚寒、产后血瘀、恶露不下者忌用。

* <a id="首乌">**何首乌**</a>：
    * 性味归经：苦、涩、甘（制熟后），性微温，入肝、肾经。
    * 功效配伍：
        * **养血**：配[熟地](#熟地)、[当归](#当归)、[白芍](#白芍)、[阿胶](#阿胶)、
        [白术](#白术)等，可治阴虚血亏以及各种贫血。
        * 益精：配[山萸肉](#山茱萸)、[山药](#山药)、[芡实](#芡实)、[五味子](#五味子)、
        [龙骨](#龙骨)、[牡蛎](#牡蛎)、[远志](#远志)、[茯苓](#茯苓)等，可用于肾虚、
        滑精、遗精、妇女带下等症。
        * **乌须发**：配[补骨脂](#补骨脂)、[当归](#当归)、[熟地](#熟地)、
        [枸杞子](#枸杞子)、[女贞子](#女贞子)、[菟丝子](#菟丝子)、[黑芝麻](#黑芝麻)、
        [旱莲草](#旱莲草)等，可治肝肾亏虚精血不足而致的须发早白。
        * 治虚性便秘：配[当归](#当归)、[肉苁蓉](#肉苁蓉)、[黑芝麻](#黑芝麻)、
        [火麻仁](#火麻仁)等，可治因年老、久病、产后失血等而津血不足从而引发的肠道
        津液缺乏、传导涩滞、大便秘结。
        * 消瘰疬痈肿：常配合[蒲公英](#蒲公英)、[紫花地丁](#紫花地丁)、[连翘](#连翘)、
        [玄参](#玄参)、生[牡蛎](#牡蛎)、[夏枯草](#夏枯草)等。
        * **止心痛**：有**强心**的作用，尤其对疲劳的心脏更明显，在治疗高血压性心脏病、
        冠心病、心绞痛等病时，可辨证添加，最好是生熟各半使用。此外，
        中医典籍说的心痛，也包括胃脘部疼痛，所以对于虚性胃脘痛也可使用。
    * 份量用法：
        * 一般为`9`～`15`g，重症时也可用至`20`～`30`g。
        * 温而不燥，补而不腻，性质平和，**适合长服**。若短期服用且欲加强功效，
        常与[熟地](#熟地)同用。
        * 何首乌有白、赤之分，二者效用大致相同，但白者补养强壮作用较赤者为弱。
        * 生何首乌适用于消瘰疬、解疮毒、通便结。
        * 制何首乌适用于补肝肾、强筋骨、养血、固精。
    * 禁忌：
        * 服后忌食猪肉、羊血、萝卜、葱、蒜。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="枸杞子">**枸杞子**</a>：
    * 性味归经：甘，平，入肝、肾经。
    * 功效配伍：
        * 滋补肝肾、益精明目：可治肝肾不足，辨证施治如下：
            * 若见腰膝无力、脐腹隐痛、阳痿不举、大便溏泄等症，可配[熟地](#熟地)、
            [山药](#山药)、[山萸肉](#山茱萸)、[肉桂](#肉桂)、[附片](#附子)、
            [鹿角](#鹿角)胶、[菟丝子](#菟丝子)等同用，例如右归丸。
            * 若精血不能上注于目而致两目昏暗、视物模糊等，可配[熟地](#熟地)、
            [山药](#山药)、[山萸肉](#山茱萸)、[茯苓](#茯苓)、[泽泻](#泽泻)、
            [菊花](#菊花)等，例如杞菊地黄丸。
        * 生津止渴：配[天冬](#天冬)、[麦冬](#麦冬)、[山药](#山药)、[玉竹](#玉竹)、
        [地黄](#生地)、[知母](#知母)等，用于消渴病。
    * 份量用法：一般为`3`～`9`g。
    * 禁忌：外邪实热、脾虚有湿及泄泻者慎用。
    * 熏硫鉴别：色泽异常鲜亮红艳，质地黏手易碎，入水易沉易褪色。

* <a id="女贞子">女贞子</a>：
    * 性味归经：甘、苦，性微凉，入肝、肾经。
    * 功效配伍：
        * 补肝肾、乌须发、清（气分）热明目：配[首乌](#首乌)、[桑椹](#桑椹)、
        [生地](#生地)、[杜仲](#杜仲)、[山药](#山药)、[旱莲草](#旱莲草)、
        [枸杞子](#枸杞子)等，可用于肝肾**阴虚**所致的发热、头发早白、两目昏花、
        耳鸣耳聋、牙齿松动、腰膝酸痛等症。
        * 治慢性虚损诸证：配[桑椹](#桑椹)、[旱莲草](#旱莲草)制成蜜丸，长服。
    * 份量用法：
        * 一般用`6`～`9`g。
        * 性质平和，补阴而不腻滞，**适合长服**，不似[生地](#生地)、
        [熟地](#熟地)般易腻滞，但滋阴之力也不如二地。
        * 胃有寒或老年，可佐用一些补脾暖胃的药，如[白术](#白术)、[陈皮](#陈皮)、
        [草豆蔻](#草豆蔻)等。
    * 禁忌：阳虚、脾胃虚寒泄泻者忌用。

* <a id="旱莲草">旱莲草</a>：
    * 性味归经：甘、酸，性平（抑或是“寒”？），入肝、肾经。
    * 功效配伍：
        * 滋补肝肾：配[女贞子](#女贞子)名“二至丸”，常用于肝肾阴虚之头发早白、
        脱发，也可在为丸时加入[桑椹](#桑椹)汁。
        * 凉血止血：
            * 用于尿血：配[生地](#生地)、[玄参](#玄参)、[白茅根](#白茅根)、[黄柏](#黄柏)炭、
            大小蓟、[瞿麦](#瞿麦)、[泽泻](#泽泻)等。
            * 用于吐血：配生[石膏](#石膏)、[知母](#知母)、[黄芩](#黄芩)、[白及](#白及)、
            [藕节](#藕节)炭等。
            * 用于便血：配[槐角](#槐角)、[地榆](#地榆)、[槐花](#槐花)炭、[黄柏](#黄柏)、
            [防风](#防风)等。
            * 用于崩漏：配[桑寄生](#桑寄生)、[续断](#续断)炭、[白术](#白术)、[棕榈](#棕榈)炭、
            [艾叶](#艾叶)炭、[当归](#当归)炭、[阿胶](#阿胶)等。
    * 份量用法：
        * 一般为`9`g，特殊需要时可用`15`～`30`g。
        * 墨旱莲偏于补肾、滋阴、止血；红旱莲偏于凉血、活瘀、清热、治疮疡。
        只写旱莲草则药房一般给前者。
    * 禁忌：脾肾虚者忌用。

* <a id="桑椹">**桑椹**</a>：
    * 性味归经：甘、酸，性凉，入心、肝、肾经。
    * 功效配伍：滋阴补血，滋补肝肾，常见配伍如下：
        * 聪耳明目：可单用本品，但量须大，例如中成药桑椹膏。
        * 治头发早白、脱落：常配合[熟地](#熟地)、[当归](#当归)、[枸杞子](#枸杞子)、
        制[首乌](#首乌)、[旱莲草](#旱莲草)、[女贞子](#女贞子)等，
        * 治阴虚血少之眩晕、失眠：配[旱莲草](#旱莲草)、[女贞子](#女贞子)。
        * 治阴虚的口渴便秘：配[麦冬](#麦冬)、[沙参](#沙参)、[玉竹](#玉竹)、
        [火麻仁](#火麻仁)、[黑芝麻](#黑芝麻)等。
        * 治肝阳上亢之高血压：配[夏枯草](#夏枯草)、[桑叶](#桑叶)、[菊花](#菊花)等。
    * 份量用法：一般为`6`～`9`g（另有说法为`9`～`15`g，因药性平和，少用不易见效，
    宜大量持续服用），熬膏可用`30`～`60`g。
    * 禁忌：脾胃虚寒泄泻或大便溏软者忌用，血虚有寒者不宜用。

* <a id="杜仲">杜仲</a>：
    * 性味归经：甘、微辛，性温，入肝、肾经（气分）。
    * 功效配伍：
        * 补肝肾、强筋骨、壮腰膝：常配[熟地](#熟地)（还能解熟地之腻滞）、
        [续断](#续断)（骨科、内科老拍档）、[怀牛膝](#怀牛膝)、[山药](#山药)、
        [山萸肉](#山茱萸)、[补骨脂](#补骨脂)等同用。
        * 安胎：可配[桑寄生](#桑寄生)（腰痛、安胎老拍档）、[续断](#续断)、[白术](#白术)、
        [熟地](#熟地)、[白芍](#白芍)、[苏梗](#苏梗)、[当归](#当归)等同用，
        治肾虚而致的胎动。
        * 降血压：可配[桑寄生](#桑寄生)、[牛膝](#牛膝)、[夏枯草](#夏枯草)等，
        治高血压而有肾虚兼症。
    * 份量用法：
        * 一般为`3`～`9`g（另有说法用`6`～`15`g，大剂量可用至`30`～`60`g）。
        * 未制过的杜仲舒筋补肝力大，治肝虚风动的头晕目眩、肝虚湿阻的下阴湿痒；
        * 盐杜仲补肾壮骨、安胎，治肾虚腰痛、阳痿滑精、胎动不安、高血压；
        * 杜仲炭补益肝肾、止血，治肝肾虚弱、冲任不固之经水不净、崩漏下血。
    * 禁忌：肾阴不足、阴虚火旺者不宜使用。

* <a id="续断">续断</a>：
    * 性味归经：辛、苦、甘，性微温，入肝、肾经（血分）。
    * 功效配伍：
        * 补肝肾：常配合[杜仲](#杜仲)、[狗脊](#狗脊)、[怀牛膝](#怀牛膝)、[生地](#生地)、
        [熟地](#熟地)、制[附片](#附子)等，可治因肾虚而致的腰痛脚软、行走不利等症。
        * 续筋骨：常配合[杜仲](#杜仲)（骨科老拍档）、[牛膝](#牛膝)、[骨碎补](#骨碎补)、
        [当归](#当归)、[川芎](#川芎)、[乳香](#乳香)、[没药](#没药)、[三七](#三七)等，
        可消肿止痛，接续筋骨。
        * 安胎：
            * 治胎动：暂略。
            * 治胎漏：暂略。
    * 份量用法：一般为`5`～`10`g，特殊时可用`25`～`30`g。
    * 禁忌：
        * 气郁者忌用。
        * 治**血病**时禁与**苦寒药**同用，包括[生地](#生地)、[黄连](#黄连)、
        [栀子](#栀子)、[雷丸](#雷丸)等。
        * 胎前禁与大辛热药同用。
    * 造假鉴别：待补充。

* <a id="莲子">**莲子**</a>：
    * 性味归经：甘、涩，性平，入心、脾、肾经。
    * 功效配伍：
        * 养心安神：对于心虚或心肾不交而导致的心神不安、失眠多梦等症，常配合[茯神](#茯苓)、
        [远志](#远志)、[柏子仁](#柏子仁)、[酸枣仁](#酸枣仁)、[珍珠母](#珍珠母)、
        [龙齿](#龙齿)等同用。
        * 健脾、厚肠胃：常配合[白术](#白术)、[山药](#山药)、[扁豆](#扁豆)、[茯苓](#茯苓)、
        [党参](#党参)、[芡实](#芡实)、[木香](#木香)等，用于治疗脾虚泄泻、消化不良、
        便溏少食、面黄白消瘦等脾虚证。
        * 补肾固涩、止泻：
            * 对于心肾虚而致的遗精，常配合[生地](#生地)、[山萸肉](#山萸肉)、
            [五味子](#五味子)、[远志](#远志)、[金樱子](#金樱子)、[锁阳](#锁阳)等同用。
            * 对于脾肾两虚、湿邪下注而致的久泻、白带过多、白浊等，常配合[肉豆蔻](#肉豆蔻)、
            [芡实](#芡实)、[白鸡冠花](#白鸡冠花)、炒[山药](#山药)、炒[薏米](#薏米)、
            [诃子](#诃子)、[白术](#白术)、[茯苓](#茯苓)、[白石脂](#白石脂)等同用。
    * 份量用法：`2`～`10`g。
    * 禁忌：气虚者慎用。

### <a id="常用药提要/理气药">10.4 理气药</a>

* <a id="陈皮">陈皮</a>：
    * 性味归经：辛、苦，性温，入脾、肺经。
    * 功效配伍：辛能散，苦能燥能泻，温能补能和，多作辅助之品，同补药则实，同泻药则泻，
    同升药则升，同降药则降，常见配伍如下：
        * 消胀止呕：由于肺胃气滞而致的胸闷、上腹胀满、恶心呕吐，可配合[枳壳](#枳壳)、
        [紫苏](#紫苏)梗、[紫苏](#紫苏)子等同用。
        * 燥湿化痰止咳：可配合[茯苓](#茯苓)、[紫苏](#紫苏)子、[杏仁](#杏仁)、
        炒[莱菔子](#莱菔子)、[旋覆花](#旋覆花)、[前胡](#前胡)等同用。注意由于脾恶湿，
        为生痰之源，所以祛湿也相当于**健脾**。
        * 理气**开胃**：对中焦气滞、食欲不振等症，可配[麦芽](#麦芽)、[谷芽](#谷芽)、
        [蔻衣](#蔻衣)、[神曲](#神曲)、[山楂](#山楂)等同用，有促进食欲的功能。
        * **避免**补药引起**壅气胀满**的副作用：在使用[党参](#党参)、[黄芪](#黄芪)、
        [白术](#白术)、[山药](#山药)、[地黄](#熟地)等补药时，加一些陈皮，可防止胸闷、
        中满、食欲不振等副作用的发生。
    * 份量用法：一般为`3`～`9`g。
    * 禁忌：本品性味香燥，过用、久用可耗散正气；阴虚燥咳、无气滞者（尤其还有气虚）勿用；
    吐血证慎用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="木香">木香</a>：
    * 性味归经：辛、苦，性温，入脾、胃、大肠、三焦经。
    * 功效配伍：
        * 行肠胃滞气：可配合[藿香](#藿香)、[香附](#香附)、[高良姜](#高良姜)、
        [槟榔](#槟榔)、**[砂仁](#砂仁)**、[草豆蔻](#草豆蔻)、[丁香](#丁香)等同用，
        治疗肠胃气滞所致的胃脘胀闷疼痛、腹胀腹痛等症。
        * 和胃健脾：用于消化不良、食欲不振、纳呆等症，可配合[党参](#党参)、
        [茯苓](#茯苓)等同用，典型处方有香砂六君汤/丸。
        * 芳香化湿：常配合[藿香](#藿香)、[佩兰](#佩兰)、[竹茹](#竹茹)、[茯苓](#茯苓)、
        [木瓜](#木瓜)、[黄柏](#黄柏)、[黄连](#黄连)等同用，治疗因肠胃气滞、
        湿停不化所致的呕吐、腹痛、泄泻等症。
        * 治疗痢疾：配[黄连](#黄连)（即“香连丸”），并可根据是否湿重、热重、
        食滞等证添加其他药物。
        * 消除补药的滋腻：例如香砂六君汤/丸、归脾汤/丸都用到木香。
    * 份量用法：
        * 一般为`0.9`～`9`g，特殊需要时也可用至`12`g。
        * 理气多生用（不宜久煎，需要**后下**），醒脾、止泻、实大肠时多煨用（不需后下）。
        * 有云木香、广木香、青木香的区别，临床一般用**广木香**，
        并且青木香与广木香不是同一种药品，不可混淆。
    * 禁忌：肺虚有热、血枯致燥者忌用，阴虚津液不足者慎用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="香附">香附</a>：
    * 性味归经：辛、微苦、微甘，性平，入肝、三焦经。
    * 功效配伍：**最常用**的**理气开郁**药，其性宣扬，能通行十二经、八脉的气分，
    前人称它能“**主一切气**”、**解六郁**（气郁、血郁、痰郁、食郁、火郁、湿郁）、
    “气病之总司”、“妇科之主帅”，还兼入血分。常用于以下几种情况：
        * **疏肝**解郁：常配合[柴胡](#柴胡)、[白芍](#白芍)、[郁金](#郁金)、[青皮](#青皮)、
        [陈皮](#陈皮)、[木香](#木香)、[厚朴](#厚朴)、[紫苏](#紫苏)梗等同用。
        兼有六郁者还需添加对应的解郁药，在此不再展开。
        * 行气定痛：最常用于气滞胃痛，常配合[高良姜](#高良姜)、[木香](#木香)、
        [白豆蔻](#白豆蔻)、[川楝子](#川楝子)、[延胡索](#延胡索)、[白芍](#白芍)、
        [紫苏](#紫苏)梗等同用。
        * 理气调经：本为行气药，但又能入血分，故称**血中气药**（有此特点的还有[当归](#当归)、
        [川芎](#川芎)），常用于治疗因肝气郁滞而引发的经期问题，常配合[当归](#当归)、
        [白芍](#白芍)、[熟地](#熟地)、[红花](#红花)、[五灵脂](#五灵脂)、[川楝子](#川楝子)、
        [乌药](#乌药)、[桃仁](#桃仁)等同用。本品还能引补血药至气分以生血，无论胎前、产后各症，
        皆可使用，又称“女科要药”。
        * 增强诸药疗效：可与补气药、补血药、理气药、解郁药、清热药、安神药、解表药、
        化痰药、活血化瘀药等配伍使用。
    * 份量用法：一般为`3`～`9`g。生用，偏于上行胸膈，外达皮肤；制熟用，偏入肝肾而利腰足。
    制法有若干种，分别侧重不同功效，在此不展开。
    * 禁忌：气虚无滞、阴虚血热者忌用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="厚朴">厚朴</a>：
    * 性味归经：辛、苦，性温，入脾、胃、大肠经。
    * 功效配伍：
        * **下/降气**：治疗胸腹胀满、气上逆而喘咳之症，配伍可参考[桂枝](#桂枝)加厚朴[杏仁](#杏仁)汤、
        苏子降气汤。
        * 燥湿消胀：可配合[木香](#木香)、[干姜](#生姜)、[草豆蔻](#草豆蔻)、[陈皮](#陈皮)、
        [茯苓](#茯苓)、[藿香](#藿香)等同用。
        * 治疗食滞腹痛：可配焦[山楂](#山楂)、[麦芽](#麦芽)、[鸡内金](#鸡内金)等。
        * 治疗久泻：配[党参](#党参)、[白术](#白术)、[茯苓](#茯苓)、[肉豆蔻](#肉豆蔻)、
        [五味子](#五味子)等。
    * 份量用法：
        * 一般为`2`～`6`g，急重症时也可用到`9`～`12`g或更多。
        * 生用偏于下气，姜汁炒用偏于止呕。
    * 禁忌：
        * 内热津枯、脾胃虚弱便溏者慎用。
        * 孕妇忌用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="砂仁">砂仁</a>：
    * 性味归经：辛，温，入脾、胃、肾经。
    * 功效配伍：温燥、行气而不破气、调中而不伤中。
        * 化湿行气：因气滞及脾胃寒湿而引起脘腹胀满、痰湿积滞、呕吐、泄泻、腹痛、
        消化不好等症时，可配合[枳实](#枳实)、[白术](#白术)、[木香](#木香)、[陈皮](#陈皮)、
        [茯苓](#茯苓)、[藿香](#藿香)、[焦神曲](#焦神曲)等同用。
        * 温脾止泻：因脾胃虚寒而致的泄泻（腹冷喜按、大便清稀），常配合[党参](#党参)、
        [白术](#白术)、[木香](#木香)、[炮姜](#炮姜)、[茯苓](#茯苓)等同用。
        * 温中安胎：因妊娠而胃气上逆、胸闷呕吐等导致的胎动不安，可配合[苏叶](#苏叶)、
        [藿香](#藿香)、[黄芩](#黄芩)、[白术](#白术)、[木香](#木香)、[当归](#当归)等同用。
        * **醒脾**、**引气归元（肾）**：重用[熟地](#熟地)等质地滋腻的补药时，配用一些砂仁，
        既可消除滋腻碍胃之弊、保持食欲，又可引熟地归肾，为此前人甚至有“砂拌熟地”
        （用砂仁拌粘在熟地上）的制作方法。另外，若要引火归元（肾），则用[肉桂](#肉桂)。
    * 份量用法：一般为`1.5`～`4.5`g，特殊需要时可用`6`～`9`g。
    入煎剂时，宜`打碎`**后下**，且久煎会减小药效，通常煎`3`～`5`分钟。
    砂壳类似，且因其体轻，份量通常为`0.9`～`1.5`g或`2`～`2.5`g。
    * 禁忌：本品芳香温燥，阴虚有热者慎用。

### <a id="常用药提要/寒凉药">10.5 寒凉药</a>

* <a id="密蒙花">密蒙花</a>：
    * 性味归经：甘，微寒，入肝经。
    * 功效配伍：**眼科常用药**，作用如下：
        * 清肝虚热：与[白蒺藜](#白蒺藜)、[菊花](#菊花)、[决明子](#决明子)、[石决明](#石决明)、
        [羌活](#羌活)、[谷精草](#谷精草)等同用，可治青盲、目昏、多泪、多眵（眼屎）、
        羞明、小儿疳气攻眼等症。
        * 明目退翳：
            * 消目中赤脉（血丝）、去翳、治久视无力，可与[木贼草](#木贼草)、[桑叶](#桑叶)、
            [夏枯草](#夏枯草)、[菊花](#菊花)、[枸杞子](#枸杞子)、[蝉蜕](#蝉蜕)、
            [夜明砂](#夜明砂)等同用；
            * 祛目中之风而治眼珠发痒，可配[蔓荆子](#蔓荆子)、[防风](#防风)、[赤芍](#赤芍)、
            [菊花](#菊花)、[荆芥](#荆芥)、[薄荷](#薄荷)（少量）等同用。
    * 份量用法：一般为`3`～`9`g。
    * 禁忌：本品有散瞳作用，故瞳孔散大者忌用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

### <a id="常用药提要/活血化瘀药">10.7 活血化瘀药</a>

* <a id="川芎">川芎</a>：
    * 性味归经：辛，温，入肝、胆、心包经。
    * 功效配伍：
        * 行气活血：与补血药同用，可防止滋腻，例如与[熟地](#熟地)、
        [当归](#当归)、[白芍](#白芍)组成经典处方“四物汤”。
        * 燥湿搜风：可治（偏）头痛和风湿痹痛，常与[荆芥](#荆芥)、[防风](#防风)、
        [白芷](#白芷)、[羌活](#羌活)、[独活](#独活)、[细辛](#细辛)、
        [菊花](#菊花)、[薄荷](#薄荷)等同用。
        * 开郁调肝：肝以辛散为顺（在肝阴肝血充足的前提下），川芎辛散可解气郁、血郁，
        常配合[香附](#香附)、[柴胡](#柴胡)、[白芍](#白芍)、[川楝子](#川楝子)、
        [当归](#当归)、[苏梗](#苏梗)、[枳壳](#枳壳)等同用。
    * 份量用法：一般为`1.5`～`9`g。
    * 禁忌：阴虚火旺、阴虚气弱、劳热多汗者忌用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。

* <a id="丹参">丹参</a>：
    * 性味归经：苦，微寒，入心、肝经，专走血分，甚至有“一味丹参饮，功同四物汤”的夸张说法。
    * 功效配伍：
        * 活瘀血：可与治疗气滞血瘀的药配合使用。与[大青叶](#大青叶)、[板蓝根](#板蓝根)、
        [蒲公英](#蒲公英)、[鳖甲](#鳖甲)、[牡蛎](#牡蛎)、[赤芍](#赤芍)、[青皮](#青皮)、
        [陈皮](#陈皮)等同用，除了能治肝郁胁痛，甚至还能用于胆囊炎、肝炎、早期肝癌化。
        * 生新血：实际上是通过祛瘀血来达到生新血的目的（不是直接生血），
        可配合[当归](#当归)、[生地](#生地)、[白芍](#白芍)、[川芎](#川芎)、
        [党参](#党参)、[白术](#白术)、[茯苓](#茯苓)等同用，尤其适合血虚而**微有热象**者。
        * 凉血、安神：
            * 有斑疹者，可配[生地](#生地)、[玄参](#玄参)、[赤芍](#赤芍)、
            [牡丹皮](#牡丹皮)、[地骨皮](#地骨皮)等同用；
            * 烦躁不眠者，可配[生地](#生地)、[黄连](#黄连)、[郁金](#郁金)、[远志](#远志)、
            [酸枣仁](#酸枣仁)、[珍珠母](#珍珠母)、[麦冬](#麦冬)等同用。
    * 份量用法：一般为`9`～`30`g。炒丹参能改善其微寒之性，适用于气血两虚但无热象者，
    养血活血；酒丹参性温且散，活力增强，用于肝脾肿大、瘀血作痛；还有其余炮制用法不在此展开。
    * 禁忌：无瘀血、月经过多、咳血尿血、孕妇慎用。
    * 造假鉴别：待补充。
    * 熏硫鉴别：待补充。
